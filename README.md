# Portifólio

Site profissional para apresentar projetos, atrair clientes e receber solicitações de orçamento.

## Sobre

Um portifólio moderno com foco em conversão: projetos reais, proposta de valor clara e formulário de contato integrado com envio de e-mail.

## Funcionalidades

- **Galeria de Projetos** - Projetos destacados em funcionamento
- **Formulário de Contato** - Solicite um orçamento ou conversa sobre seu projeto
- **Envio de E-mail com SendGrid** - Notificação automática de novos contatos
- **Painel Administrativo** - Gerenciamento de contatos (restrito)

## Tecnologias

- Next.js
- TypeScript
- Prisma
- Tailwind CSS
- SendGrid
