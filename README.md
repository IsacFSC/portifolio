# Portfólio

Este é o meu site oficial para divulgar meu trabalho e experiência.

## Sobre

Um portfólio moderno onde você pode ver exemplos de projetos em produção e conhecer os serviços que ofereço.

## Funcionalidades

- **Galeria de Projetos** - Projetos destacados em funcionamento
- **Formulário de Contato** - Solicite um orçamento ou conversa sobre seu projeto
- **Painel Administrativo** - Gerenciamento de contatos (restrito)

## Tecnologias

- Next.js
- TypeScript
- Prisma
- Tailwind CSS
